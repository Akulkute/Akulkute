## 🚀 Projects

🚁 **Custom Bicopter Flight Controller — SoC-Level Embedded Firmware**
C · STM32F411 (ARM Cortex-M4) · FreeRTOS · I2C · PWM | Ongoing
* Engineered a 1kHz real-time firmware control loop on STM32F411 with a custom PID controller in C for dual-rotor stabilization
* Integrated MPU6050 IMU and BMP280 barometer via I2C for attitude estimation and altitude hold
* Programmed high-frequency PWM for differential thrust control, servo-actuated pitch/yaw vectoring, and SBUS/CRSF radio protocol decoding
* Implemented a hardware-level safety failsafe, demonstrating robust SoC-level embedded firmware design

🤖 **VAANI — Assistive Communication Robot**
ESP32 · STT/TTS · WebSocket · Node.js · Flask · Hardware-Software Co-design | 6 Months
* Developed an ESP32-based real-time speech-to-speech communication device using INMP441 microphone and Whisper/Vosk STT/TTS engine
* Enabled offline AI inference directly on embedded hardware without internet dependency
* Integrated WebSocket-based audio streaming pipeline, Node.js/Flask backend, and TFT display with servo motors
* 🏆 Design Patent Filed

📡 **Mine Guard — IoT Gas & Labour Health Monitoring System**
C · ESP-NOW · IoT · Custom PCB · Wireless Sensing | 3 Months
* Engineered an ESP-NOW based IoT device detecting mine gases (CH4, CO, CO2, SOx, NOx) and monitoring worker vitals in real time
* Achieved sub-50ms transmission latency to a central dashboard with no local Wi-Fi required
* Designed custom PCB and firmware stack from scratch, optimized for low-power operation in interference-prone environments

🐁 **Micromouse — Maze-Solving Autonomous Robot**
Teensy 4.1 · C++ · TOF Sensors · Flood-Fill Algorithm | 2 Months
* Built a fully autonomous maze-solving robot using Teensy 4.1, 5 TOF distance sensors, and encoder motors with TB6612FNG driver
* Implemented a software-based Flood-Fill algorithm improving path discovery efficiency from 60% to 90%
